# 🎬 Movie-Deck

Movie-Deck is a simple **front-end movie display web page** made using **HTML & CSS**.

This project shows a movie slider section with movie cards and is created for learning UI design and layout using HTML and CSS.

---

## 📌 About The Project

This page displays a list of popular movies (with ratings, posters, and descriptions).

There is **no backend or database** — it is a static web page.

The content is directly written inside the HTML file.

---

## 🛠 Technologies Used

- 🧱 HTML — for page structure  
- 🎨 CSS — for design and styling
